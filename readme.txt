== Description ==

Intelligent WordPress plugin which helps you automate image file renaming for better SEO.

With Seopic, you can easily optimize your website's images and boost your search engine ranking. The plugin uses intelligent algorithms to generate SEO-friendly file names based on the alt text of your media, making it a breeze to improve your website's SEO.

Seopic supports all default WordPress Gutenberg media blocks - <strong>Image</strong>, <strong>Gallery</strong>, <strong>Cover</strong>, and <strong>Media & Text</strong> block, providing a comprehensive solution for optimizing your WordPress images. The plugin also renames media files of <strong>Featured Images</strong> of posts. Say goodbye to manual renaming and hello to an effortless and intelligent way of optimizing your images.

Plugin is fully compatible with all popular image formats as <strong>JPEG</strong>, <strong>SVG</strong>, <strong>WEBP</strong>, <strong>PNG</strong> and <strong>GIF</strong>, and is updated regularly to stay ahead of the latest SEO trends and algorithms.

Whether you're a website owner, blogger, or e-commerce business, Seopic is the perfect solution for optimizing your images for search engines and achieving the best possible results.

<strong>HOW IT WORKS</strong>

<strong>Image Renaming:</strong> Seopic automatically renames your images when the alt text is edited or added to a default WordPress block, such as Image, Gallery, Cover, or Media & Text.

<strong>Media Library:</strong> The original image is saved in your media library, and a new copy of the image with a newly generated, optimized file name is created. This means you will never lose your original images.

<strong>Path Updating:</strong> Seopic then automatically updates the path of your optimized images in the post or page content, ensuring that all of your images are properly linked and optimized for search engines.

<strong>Ongoing Optimization:</strong> Seopic continues to optimize your images as you add, edit, or update content on your website, ensuring that your images are always optimized for the best possible search engine results.

== Installation ==

Install the plugin either via the WordPress plugin directory by searching the plugin's name "Seopic" or by uploading the files to your server at wp-content/plugins.

== Frequently Asked Questions ==

= What is Seopic? =
Seopic is a free WordPress plugin that optimizes your images for search engines. It automatically generates <strong>optimized file names</strong> for your images and updates the path of your images in the post or page content.

= Is Seopic easy to use? =
Yes, Seopic requires no technical knowledge or experience with SEO to use. Simply install the plugin, and let Seopic handle the rest.

= How does Seopic work? =
Seopic renames your images when <strong>the alt text is edited or added</strong> to a default WordPress media block, such as <strong>Image</strong>, <strong>Gallery</strong>, <strong>Cover</strong>, or <strong>Media & Text</strong>. It also supports <strong>Featured Images</strong> of posts. The original image is saved in the media library, and a new copy of the image with a newly generated, optimized file name is created. Seopic then automatically updates the path of your optimized image in the post or page content.

= Does Seopic support all image formats? =
Yes, Seopic supports all popular image formats, including <strong>JPEG</strong>, <strong>SVG</strong>, <strong>WEBP</strong>, <strong>PNG</strong>, <strong>GIF</strong>, and more.

= What is the file format Seopic generates? =
The plugin uses for the file name format the alt text of the media, ending with a random 14 chars. For example, if your media has alt text <strong>"Grassy hills near mountains"</strong> then the generated file name can have format <strong>grassy-hills-near-mountains-bmu7yhxaluhkhm.jpg</strong>.

= What are the benefits of using Seopic? =
Seopic improves your search engine rankings by optimizing your images for search engines, increasing your visibility and driving more traffic to your website. It also saves you time by <strong>automating the image optimization process</strong> and is easy to implement and manage.

= Can I still access my original images after using Seopic? =
Yes, Seopic <strong>saves the original images</strong> in your media library, so you can still access them if needed.

= Why does it take longer to rename images with Seopic? =
The time it takes to rename images with Seopic depends on two factors: <strong>the number of images</strong> in a post or page, and <strong>the size of the image files</strong>. If a post or page contains a large number of images or if the image files are large in size, it may take longer to rename all of the images. However, Seopic is designed to be fast and efficient, and in most cases, the image renaming process should be quick and seamless.

= Is Seopic regularly updated? =
Yes, Seopic is updated regularly to stay ahead of the latest SEO trends and algorithms, ensuring that your images are always optimized for the best possible results.
